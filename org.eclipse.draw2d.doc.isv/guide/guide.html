Draw2d includes the following packages:

  • org.eclipse.draw2d - the majority of the interfaces and implementation can be found in this package

  • org.eclipse.draw2d.geometry - provides utility classes for representing and working with geometric data

  • org.eclipse.draw2d.graph - stand-alone graph drawing algorithms such as directed graph layout

  • org.eclipse.draw2d.parts - special purpose figures that are used as-is, including a figure that generates a thumbnail

  • org.eclipse.draw2d.text - specialized figures and layouts providing rich text capabilities

  • org.eclipse.draw2d.widgets - custom SWT controls which utilize figures for painting
